## 아이디어 1: 여행 일지 감성 (Analog Travel Journal)
<response>
<text>
**Design Movement**: 아날로그 여행 일지 + 현대적 미니멀리즘

**Core Principles**:
- 손으로 쓴 듯한 따뜻함과 디지털 정확성의 공존
- 여행의 설렘을 담은 따뜻한 색감
- 카드 기반의 레이아웃으로 각 지출이 '여행 메모'처럼 느껴짐
- 여백을 통한 호흡감

**Color Philosophy**:
- 주색: 따뜻한 테라코타 (#C17B5A) - 여행의 따뜻함
- 보조색: 크림 화이트 (#FAF7F2) - 일지의 종이 느낌
- 강조색: 딥 네이비 (#1E3A5F) - 신뢰와 정확성
- 배경: 아이보리 (#F5F0E8)

**Layout Paradigm**:
- 좌측 사이드바에 여행 프로젝트 목록
- 우측 메인 영역에 날짜별 타임라인 형태의 지출 목록
- 상단에 여행 날짜 버튼들이 수평 스크롤로 배치

**Signature Elements**:
- 둥근 날짜 버튼 (여행 날짜별 동그란 배지)
- 카드에 카테고리별 색상 왼쪽 보더
- 손글씨 느낌의 폰트 (Noto Serif KR)

**Interaction Philosophy**:
- 날짜 버튼 클릭 시 부드러운 스크롤
- 카드 호버 시 살짝 들어올려지는 효과
- 정산 완료 시 체크 애니메이션

**Animation**:
- 페이지 진입 시 카드들이 순차적으로 페이드인
- 금액 입력 시 숫자 카운트업 애니메이션
- 정산 버튼 클릭 시 confetti 효과

**Typography System**:
- 제목: Noto Serif KR (700) - 따뜻하고 고급스러운 느낌
- 본문: Pretendard (400/500) - 가독성
- 금액: 모노스페이스 느낌의 굵은 숫자
</text>

## 아이디어 2: 모던 핀테크 대시보드 (Modern Fintech Dashboard)
<response>
<text>
**Design Movement**: 모던 핀테크 + 다크 모드 우선 디자인

**Core Principles**:
- 데이터 중심의 명확한 시각화
- 다크 배경에 네온 강조색으로 현대적 느낌
- 정보 밀도와 가독성의 균형
- 그래프와 숫자가 주인공

**Color Philosophy**:
- 배경: 딥 차콜 (#0F1117) - 집중력
- 카드: 다크 그레이 (#1C1F2E) - 깊이감
- 주색: 에메랄드 그린 (#00D4AA) - 금융 성공
- 강조: 코랄 (#FF6B6B) - 지출 경고
- 텍스트: 순백 (#FFFFFF) / 그레이 (#8B8FA8)

**Layout Paradigm**:
- 상단 대시보드 통계 카드 (총 지출, 내 부담금, 정산 현황)
- 중앙 그래프 영역
- 하단 지출 목록 테이블

**Signature Elements**:
- 글로우 효과가 있는 금액 표시
- 진행률 바로 표시되는 정산 현황
- 그라데이션 차트

**Interaction Philosophy**:
- 호버 시 데이터 툴팁
- 클릭 시 상세 드릴다운
- 실시간 합계 업데이트

**Animation**:
- 숫자 카운트업 애니메이션
- 차트 드로잉 애니메이션
- 카드 shimmer 로딩 효과

**Typography System**:
- 숫자: JetBrains Mono (700) - 정확한 데이터 표현
- 제목: Pretendard (600/700)
- 본문: Pretendard (400)
</text>

## 아이디어 3: 여행 지도 감성 스크랩북 (Travel Scrapbook)
<response>
<text>
**Design Movement**: 스칸디나비안 미니멀 + 여행 스크랩북

**Core Principles**:
- 깔끔하고 공기처럼 가벼운 레이아웃
- 색상보다 형태와 공간으로 구분
- 기능성을 미학으로 승화
- 모든 요소가 목적을 가짐

**Color Philosophy**:
- 배경: 순백 (#FFFFFF)
- 주색: 인디고 (#4F46E5) - 신뢰와 여행
- 보조: 앰버 (#F59E0B) - 따뜻한 강조
- 카테고리별: 파스텔 팔레트 (식비=연두, 교통=파랑, 숙박=보라, 관광=주황)
- 텍스트: 다크 슬레이트 (#1E293B)

**Layout Paradigm**:
- 상단 여행 날짜 타임라인 (동그란 버튼들)
- 좌측 사이드패널: 멤버 목록 + 정산 현황
- 우측 메인: 지출 카드 그리드
- 플로팅 추가 버튼

**Signature Elements**:
- 카테고리별 아이콘 + 파스텔 배경 뱃지
- 멤버 아바타 (이니셜 기반 색상)
- 정산 화살표 시각화

**Interaction Philosophy**:
- 드래그로 날짜 이동
- 스와이프로 지출 삭제
- 탭으로 뷰 전환 (목록/그래프/정산)

**Animation**:
- 페이지 슬라이드 전환
- 카드 추가 시 스프링 애니메이션
- 정산 완료 시 체크 애니메이션

**Typography System**:
- 제목: Noto Sans KR (700/800) - 강렬한 대비
- 본문: Noto Sans KR (400/500)
- 금액: Pretendard (700) - 숫자 가독성
</text>

</response>
